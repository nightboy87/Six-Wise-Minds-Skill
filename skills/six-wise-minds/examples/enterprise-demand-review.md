# Example: Enterprise Demand Review

## User input

业务部门提出一个 RPA 需求，希望自动把 MDM 数据填入 MES。帮我判断这个需求是否值得进入试点。

## Expected behavior

- Trigger deep mode.
- Risk level L1 or L2 depending on operational impact.
- White Hat lists missing data: volume, exception rate, audit requirements, rollback plan.
- Black Hat checks data quality, account permission, process changes, exception handling.
- Yellow Hat identifies labor saving and standardization value.
- Green Hat proposes pilot scope.
- Blue Secretary checks whether missing facts prevent a strong conclusion.
- Chair recommends pilot / defer / split / require more info.
