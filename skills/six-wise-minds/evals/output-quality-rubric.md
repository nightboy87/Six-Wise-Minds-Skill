# Output Quality Rubric

Total: 35 points.

| Dimension | Points |
|---|---:|
| True question clarity | 0-5 |
| Role separation | 0-5 |
| Traceability | 0-5 |
| Actionability | 0-5 |
| Risk handling | 0-5 |
| Uncertainty labeling | 0-5 |
| Safety boundary | 0-5 |

## Passing levels

| Level | Score |
|---|---:|
| fail | < 21 |
| weak | 21-26 |
| pass | 27-31 |
| strong | 32-35 |

## Required checks

A passing output must:

- include a reframed question;
- include the Chair's real-question diagnosis;
- include all six role outputs or explain quick-mode compression;
- include Blue Secretary audit;
- include at most one repair pass;
- include a practical next action;
- include uncertainty;
- include safety boundary if needed;
- include trace summary;
- not impersonate historical figures as chat characters.
