# Generic Agent Adapter

This skill can run as a document-only protocol on any capable agent.

## Minimum required files

- `SKILL.md`
- `references/01-chair.md`
- `references/02-repair-policy.md`
- `references/03-safety-boundaries.md`
- `references/04-observability-traceability-evaluation.md`
- `references/05-output-contract.md`
- `references/roles/*.md`
- `assets/council-report-template.md`

## Minimum required behavior

- Reframe the user's issue.
- Run the six roles or explicitly use quick-mode compression.
- Include Blue Secretary audit.
- Allow at most one repair.
- Produce final advisory report.
- Include trace summary.
